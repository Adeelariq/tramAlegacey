# Tram A Legacy

Tram A Legacy is a premium e-commerce platform for handcrafted copper utensils and raw copper materials, based in Srinagar, Kashmir. Built with a modern tech stack, this application provides a seamless shopping experience for customers and a robust management system for administrators.

## 🛠 Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** (with custom copper theme)
- **Supabase** (Postgres + Auth + Storage)
- **Framer Motion** (animations)
- **Lucide React** (icons)


## 🌟 Key Features

- **Premium User Experience**: Immersive animations, copper-themed aesthetic, and responsive design tailored to highlight artisan craftsmanship.
- **Product & Category Management**: Dynamic filtering and viewing of products and categories.
- **Inquiry System**: Direct integration with WhatsApp and Email for product inquiries.
- **Admin Dashboard**: Secure, authenticated portal for managing inventory, categories, and featured items.
- **SEO Optimized**: Dynamic sitemaps, semantic HTML, and extensive OpenGraph metadata.
